# 上游 DTO 指针类型规范

> **级别**：零容忍
> **创建日期**：2026-05-13

---

## 规则

对于从客户端 JSON 解析后再 re-marshal 到上游供应商的请求结构体（relay/convert 路径）：

**可选标量字段必须用指针类型 + `omitempty`**

---

## 正确示例

```go
type ChatRequest struct {
    Model       string   `json:"model"`                          // 必填，非指针
    Temperature *float64 `json:"temperature,omitempty"`          // 可选
    TopP        *float64 `json:"top_p,omitempty"`                // 可选
    MaxTokens   *int     `json:"max_tokens,omitempty"`           // 可选
    Stream      *bool    `json:"stream,omitempty"`               // 可选
}
```

---

## 语义

| 客户端 JSON | Go 值 | Marshal 结果 |
|------------|-------|-------------|
| 字段缺失 | `nil` | 字段省略 |
| 显式设为 0/false | 非 nil 指针 | 字段保留（发送给上游） |

---

## 禁止

```go
// ❌ 非指针 + omitempty 会丢失显式零值
type ChatRequest struct {
    Temperature float64 `json:"temperature,omitempty"` // 客户端发 0.0 会被丢弃！
    Stream      bool    `json:"stream,omitempty"`      // 客户端发 false 会被丢弃！
}
```

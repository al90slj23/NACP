# Git 提交与分支规范

> **创建日期**：2026-05-13

---

## Commit 格式

```
<type>(<scope>): <subject>
```

### 类型

| 类型 | 说明 |
|------|------|
| `feat` | 新功能 |
| `fix` | 修复 Bug |
| `enhance` | 增强现有功能 |
| `refactor` | 重构（不改变行为） |
| `perf` | 性能优化 |
| `docs` | 文档变更 |
| `chore` | 构建/工具变更 |

### Scope 建议

| Scope | 说明 |
|-------|------|
| `relay` | 中继/代理核心 |
| `retry` | 重试机制 |
| `channel` | 渠道管理 |
| `model` | 数据模型 |
| `web` | 前端 |
| `auth` | 认证授权 |
| `billing` | 计费 |

---

## 分支规范

| 类型 | 格式 | 示例 |
|------|------|------|
| 主分支 | `main` | — |
| 功能 | `feature/<name>` | `feature/smart-retry` |
| 修复 | `fix/<name>` | `fix/stream-timeout` |
| 增强 | `enhance/<name>` | `enhance/retry-backoff` |
| 发布 | `release/classic-plus-<ver>` | `release/classic-plus-0.2.0` |

---

## 版本号

格式：`classic-plus-<major>.<minor>.<patch>`

- major: 重大架构变更
- minor: 新功能
- patch: bugfix / 小改进

## 对设计的影响

1. **不需要特殊处理窗口 2** — 实际不存在
2. **智能重试可覆盖 100% 的流式错误** — 全部发生在 HTTP 状态码阶段
3. **400 错误占 75%** — 这些是参数错误，重试同一渠道无意义，应直接切换渠道
4. **503/502 占 20%** — 这些是临时性服务不可用，探测后可能恢复
5. **错误集中在少数渠道** — 健康状态机对这些渠道效果最明显
